About This application is a chess engine hobby project. The project started as a fun challenge where I wanted to challenge my colleague's chess AI by creating my own. I have since then continued to develop it after leaving it untouched for the better part of a year.

The chess engine itself is written in the Rust programming language, and it is my first big project using Rust. The GUI is written using Angular and Tauri. I chose Angular due to my familiarity with it, and I chose Tauri because I wanted to try it out as a replacement to Electron. Everything is subject to change. I hope in the future to implement an AI using neural networks.
